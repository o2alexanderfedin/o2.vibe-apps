| Boundary | Description |
|----------|-------------|
| IDB → React state | customThemeIndex and custom:* values read from IDB and parsed — malformed JSON must not crash the provider |
| localStorage → AnyThemeName | readStoredOsTheme() accepts "custom:*" strings — these come from the same localStorage this code wrote, but the key must not be allowed to bypass to an invalid theme state |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-22-01 | Tampering | refreshCustomThemes JSON.parse path | mitigate | Wrap each JSON.parse in try/catch; skip malformed entries; never set an undefined value in customThemesState |
| T-22-02 | Tampering | readStoredOsTheme returning arbitrary "custom:*" string | accept | The "custom:*" value is only used as a map lookup key into customThemesState; if absent, VIBE_THEMES["aurora"] is the fallback — no code injection possible via this path |
| T-22-03 | Information Disclosure | broadcastTheme leak of localStorage/API key | accept | broadcastTheme only sends vars (12 CSS custom property values); the API key is never in this map; same mitigation as Phase 20 |
