    Detection algorithm (two scanners, OR-combined):

    Scanner A — hex literals:
    - Match: /#([0-9a-fA-F]{3,8})\b/g
    - For 3-digit #rgb: expand to 6-digit by doubling each nibble; flag if R≠G or G≠B
    - For 6-digit #rrggbb: parse r=parseInt(hex.slice(0,2),16), g=parseInt(hex.slice(2,4),16), b=parseInt(hex.slice(4,6),16); flag if r≠g or g≠b
    - For 8-digit #rrggbbaa: parse first 6 digits same as 6-digit case; flag if r≠g or g≠b
    - For any other length (4, 5, 7): skip (unusual formats, avoid false positives)
    - Grayscale means r===g && g===b — do NOT flag

    Scanner B — rgb()/rgba() literals:
    - Match: /\brgba?\s*\(\s*(\d+)\s*,\s*(\d+)\s*,\s*(\d+)/g
    - Parse r, g, b as integers
    - Flag ONLY if r!==g || g!==b (non-grayscale)
    - rgba(0,0,0,α) and rgba(255,255,255,α) are grayscale — do NOT flag
    - The alpha channel (4th parameter) is irrelevant for this check

    No external dependencies — pure function using only regex and string ops.
